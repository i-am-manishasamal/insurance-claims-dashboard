# 📊 Insurance Claims Analytics Dashboard

> An interactive Streamlit dashboard analyzing **1,000 auto insurance claims** with **17 charts across 8 thematic sections** — built to demonstrate how insurance companies can use data to spot fraud, understand claim patterns, and drive operational decisions.

## 🚀 Live Demo

**🔗 [Click here to view the live dashboard](https://manisha-insurance-dashboard.streamlit.app/)**

## 📋 Overview

This dashboard answers key business questions for an auto insurance provider:

- Where are fraud risks concentrated?
- Which customer profiles drive the most claims?
- How do claims vary by region, time, and incident type?
- What are the operational opportunities for cost reduction?

**🔍 Key insight:** ~25% of claims in this dataset were flagged as fraud — concentrated in specific incident types and customer segments.

## ✨ Features

### 17 Interactive Charts Across 8 Sections

| Section | Charts |
|---|---|
| 📈 Key Metrics | Top-line KPIs (claims, average amount, fraud rate, customer age) |
| 🔍 Claims Breakdown | By incident type and severity |
| 💰 Financial Analysis | Claim amount distribution, by state |
| 🚨 Fraud Insights | Fraud rate by incident type, time trends |
| 👥 Customer Demographics | Gender, education, age distribution |
| 🚗 Vehicle Analysis | Top vehicle makes, claim by vehicle age |
| ⏰ Time Patterns | Incidents by hour of day, day of week |
| 💼 Claim Type Breakdown | Injury vs property vs vehicle, vehicles involved |
| 👔 Occupation & Lifestyle | Top occupations and hobbies in claims |
